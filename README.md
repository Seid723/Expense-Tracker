# Expense-Tracker
## Expense Tracker Web Application

This is a full-stack web application built with ASP.NET Core MVC that allows users to manage personal income and expenses.

### Features
- Add, edit, delete income and expense records
- View financial overview dashboard
- Structured MVC architecture
- Database integration with Entity Framework Core

### Technologies Used
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Razor Views
- Bootstrap
